SharePoint Email Column

Add a brand new column to your SharePoint, Email Column. The field in this type can save your time from filling the @domain information. For example, when support@hotmail.com is what you need to type in for this field, you only need to enter support and chose the support@hotmail.com from the options we prepared for you.

Creation

Without SharePoint Email Column, administrators have to create a “Single line of text” field for the email information; however, with this new field type, they can easily create an email column by selecting the “Email” on page “Create Column”. Comparing to the “Single line of text” column, “Email” column provides users the features of Domain Information Auto-completion and Email Address Validation.

Settings

SharePoint Email Column provides administrators an option to set a default email domains’ list. They can easily set a list of domain choices for their users to choose from and thus save the users’ time. The plugin also allows administrators to restrict the users from entering other domain information that is not included in the list they created to ensure they only get the exact email information that is really needed.

Domain Information Auto-completion

When you keep typing in, this email field will show a list of domain choices for you. You can choose the one you want by moving the focus with the "Up" and "Down" button on your keyboard.

Email Address Validation

This plugin supports the feature of Email Address Validation which guarantees only the email address in the right format can be saved.
