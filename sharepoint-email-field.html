SharePoint Email Column v1.0.0

Download

The SharePoint Email Field Auto Completion is a handy plugin to save your time from filling the @domain information, for instance, when enter support@hotmail.com, you only need to enter support@, and the SharePoint Email Field Auto Completion dropdown feature will show a list of domain choices for you to choose from.

Creation

Without SharePoint Email Filed Auto Completion, administrators have to create a “Single line of text” field for the email information; however, using this plugin, they can easily create an email column by selecting the “Email” field in the menu of “Create Column” page.

Comparing to the “Single line of text” field, “Email” field provides users the features of Domain Information Auto-completion and Email Address Validation.

Settings

SharePoint Email Field Auto Completion provides administrators an option to set a default email domains’ list. They can easily set a list of domain choices for their users to choose from and thus save the users’ time.

The plugin also allows administrators to restrict the users from entering other domain information that is not included in the list they created to ensure they only get the exact email information that is really needed.

Domain Information Auto-completion

The SharePoint Email Field Auto Completion dropdown feature will show a list of domain choices for the end users to choose from. They no longer need to enter the @domain information.

Email Address Validation

This plugin supports the feature of Email Address Validation which guarantees only the email address in the right format can be saved.
