SharePoint List View Filter v1.0.0

Download

With this SharePoint List View Filter you can easily and effectively filter items in a SharePoint List View.

Because SharePoint out of the box view filter feature is very weak, it only supports users to order the items by descending or ascending, or filter by a quite general list of options, what’s more, the filter condition is only applied to the current page of the SharePoint list view, most people may become unglued when it comes to filtering some specific information from a large amount of items that are displayed in a SharePoint list view. Then the question arises: how to filter items easily and effectively? Here we introduce you SharePoint List View Filter add-on to solve this issue.

Text Field

For Text field, SharePoint List View Filter web part provides you three filter conditions to set up, they are Starts With, Contains and Equals With, for example, presuming that you want to see items that are starts with “” and contains “”, just type these information in the blank, then all the related items within the list view will be displayed.

Numeric Field

For Numeric fields, SharePoint List View Filter add-on offers users three types of filter conditions that are Equals With, Not Equals With and From “To”. For instance, one users need to find items that equals with “” and not equals with “”, fill in the blank, all the items needed will be listed.

Date Field

Regarding to Date fields, now users have more control than before in filtering with Today, This Week, This Month, Last Week, Last Month and From “To”. Assuming that a salesman need to check the sales of Last Week, using SharePoint List View Filter, he can get all the items with a single click.

What’s more, SharePoint out of the box view filter feature is only applied to the current page of the List View; however, the filter conditions of SharePoint List View Filter are applied to the whole list View. And the SharePoint List View Filter is fully compatible with the SharePoint out of the box View feature.

How to Set Up This Feature?

  1. Go to List Setting
  2. Click List View Filter Settings (Powered by Blue Power)
  3. Set up the filter conditions according to your requirements
