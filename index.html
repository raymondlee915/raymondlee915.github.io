SharePoint Email Column

We think it is a bad idea to use a single-line text to hold the email address for your end users, because this type of field is just not for email addresses. That's why we created this product, to make your email field a real email field. There is no need to write a regular expression to validate the end users' input, and they can get some options to choose when end users typing in characters.

SharePoint List View Filter

The SharePoint List View Filter is a handy plugin to save your time from finding the items you want from one data list. This plugin provides you very filters like the ones in Office Excel. Foe example, you can find the itmes by a numeric range, or a date range. With these Excel-like filters, you can forget about creating more views in your SharePoint list.
